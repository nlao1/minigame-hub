Class descriptions:
	Game - GUI and contains the code for opening each game
	CheckersBoard - A class containing a GUI view of the Checkers Game (like in MVC)
	ConnectFourBoard - A class containing a GUI view of Connect Four (like in MVC)
	Checkers/ConnectFourTest - classes that contain JUnit test cases for the game model/logic
	Checkers - The implementation for the Checkers game (model in MVC). Controls the internal
	state of the game and is affected by the GUI controller.
	ConnectFour - The implementation for the ConnectFour game (model in MVC). Controls the internal
	state of the game and is affected by the GUI controller.
	CheckersMove - Class containing information about moves in Checkers
